# Data

This folder stores placeholder datasets and documentation for thin-film ML experiments.

- `placeholder_film_data.csv`: Example rows for development and wiring up pipelines.

The schema follows the shared feature and target list used across models:

**Feature inputs**
- Withdrawal Speed
- Dwell Time (s)
- Substrate Type
- PDMS Concentration
- Solvent Type
- Etc.

**Target outputs**
- Total Film Thickness (μm or nm)
- Bonded Film Thickness (μm or nm)
